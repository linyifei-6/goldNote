# 黄金持仓分析 - 微信小程序

## 项目说明

这是一个黄金持仓分析的微信小程序，帮助用户记录和管理黄金交易，计算持仓成本、收益等数据。

## 功能特性

### 1. 持仓分析页面
- 显示当前持仓克数
- 显示持仓成本均价（移动加权平均法）
- 显示累计已实现收益
- 显示持仓浮动收益（需输入当前市价）
- 显示总收益和总收益率

### 2. 交易页面
- 支持买入和卖出操作
- 自动计算卖出手续费（0.4%）
- 支持多个交易平台（招商、民生、浙商、支付宝、工行、微信）
- 自动生成交易ID
- 实时显示交易金额和费用预览

### 3. 历史记录页面
- 今日统计：查看当天的买入/卖出均价
- 时段统计：选择日期范围查看均价
- 自选记录统计：手动选择多条记录计算均价

## 数据结构

单条交易记录格式：
```json
{
  "id": "TX20240615001",
  "type": "buy/sell",
  "price": 1138.35,
  "weight": 20.0,
  "platform": "招商",
  "date": "2024-06-15",
  "fee_rate": 0.004,
  "fee_amount": 9.11,
  "net_amount": 22757.89,
  "timestamp": "2024-06-15 14:30:00"
}
```

## 核心计算逻辑

### 持仓成本均价
使用移动加权平均法：
```
新持仓成本 = (原持仓×原成本 + 本次买入金额) / 新持仓
```

### 卖出收益
```
卖出收益 = 卖出克数 × (成交价 × 0.996 - 持仓成本)
```

### 持仓浮动收益
```
持仓浮动收益 = 持仓克数 × (当前市价 - 持仓成本)
```

### 总收益率
```
总收益率 = (累计收益 + 持仓收益) / 总投入本金 × 100%
```

## 使用方法

1. 在微信开发者工具中导入项目
2. 配置小程序AppID（测试可使用测试号）
3. 添加TabBar图标文件到 `images/` 目录：
   - `home.png` (81x81px)
   - `home-active.png` (81x81px)
   - `transaction.png` (81x81px)
   - `transaction-active.png` (81x81px)
   - `history.png` (81x81px)
   - `history-active.png` (81x81px)

## 注意事项

1. 数据存储使用微信小程序本地存储，清除缓存会丢失数据
2. 卖出操作会校验持仓数量，不足时会提示
3. 日期默认选择当天，可手动修改
4. 建议定期备份交易记录
